# 🧠 Base de Conocimiento - FitManager Pro

### 🎯 Propuesta de Valor
Sistema SaaS integral para gimnasios que centraliza socios, pagos, accesos y agenda. Menos papeles, más resultados.

### 📦 Módulos Principales
- **Dashboard:** KPIs, ingresos, socios activos y morosos en tiempo real.
- **Gestión de Socios:** Registro ilimitado y migración de datos incluida.
- **Pagos:** Control de cobros y deudas.
- **Acceso:** Control por DNI.
- **Staff:** Múltiples usuarios para empleados.
